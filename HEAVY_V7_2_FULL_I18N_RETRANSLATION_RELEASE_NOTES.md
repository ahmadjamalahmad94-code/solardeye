# Heavy v7.2 — Full i18n Retranslation System

## هدف الإصدار
تحويل الموقع من ترجمة جزئية داخل القوالب إلى نظام ترجمة مركزي قابل للتوسع، مع إعادة ترجمة واسعة لكل واجهات الإدارة وبوابة المشترك.

## ما تم تنفيذه

### 1. قاموس مركزي للترجمة
أضيف الملف:

```text
app/services/translations.py
```

ويحتوي على قاموس عربي → إنجليزي مركزي بعدد كبير من مصطلحات وعبارات الواجهة. يمكن لاحقًا إضافة لغات جديدة عبر توسيع `TRANSLATIONS` بنفس المفاتيح.

### 2. طبقة i18n موحدة
تمت إعادة بناء:

```text
app/services/i18n.py
```

الطبقة الجديدة توفر:

- `t()` و `tr()` داخل القوالب.
- فلتر Jinja باسم `tr`.
- حفظ اللغة في الجلسة.
- الحفاظ على query params عند تغيير اللغة.
- ترجمة flash messages.
- ترجمة legacy hard-coded HTML عند عرض الموقع بالإنجليزية.
- حماية script/style/code/textarea من الترجمة حتى لا تتلف JSON أو JavaScript.

### 3. ترجمة Client-side
تم ربط القاموس المركزي مع JavaScript عبر:

```text
window.SOLARDEYE_I18N
```

وتم تعديل `app/static/js/app.js` حتى يستخدم قاموس السيرفر ويترجم النصوص الديناميكية المعروفة دون حذف النصوص العربية التي قد تكون محتوى مستخدم فعليًا.

### 4. منع تكسير الكلمات العربية
تم استخدام استبدال آمن بحدود الكلمات العربية حتى لا تتحول كلمات مثل "منصة" إلى "Fromصة".

### 5. Cache busting
تم تحديث CSS/JS إلى:

```text
v=7.2
```

### 6. تحسينات إضافية
- ترجمة أوسع لصفحات الطاقة والبطارية والتقارير والإشعارات والدعم والإدارة.
- ترجمة labels داخل أزرار OAuth وواجهات Telegram/SMS.
- ترجمة رسائل النظام والـ flash الشائعة.
- ترجمة role badges في صفحة المستخدمين.
- تحديث نصوص الإصدار في الشريط الجانبي إلى v7.2.

## ملاحظات مهمة
- هذا الإصدار يضع البنية الصحيحة للترجمة الكاملة. عند مراجعة كل صفحة لاحقًا، أي عبارة ناقصة يمكن إضافتها إلى `app/services/translations.py` بدون إعادة تصميم الصفحة.
- النصوص التي يكتبها المستخدمون بالعربية، مثل رسائل الدعم أو أسماء الأجهزة، لا يتم حذفها تلقائيًا حتى لا نخرب محتوى المستخدم.

## فحص بعد النشر
1. افتح لوحة الإدارة بالعربي.
2. اضغط English وتأكد أن الصفحة تتحول وتبقى اللغة عند التنقل.
3. افحص الصفحات التالية:
   - Dashboard
   - Subscribers
   - Support Command Center
   - Portal Support
   - Notifications
   - Reports
   - Live Data
   - Devices
4. أي عبارة واجهة ما زالت عربية في وضع English أضفها إلى `translations.py` أو أرسلها للمراجعة ضمن الجولة القادمة.
