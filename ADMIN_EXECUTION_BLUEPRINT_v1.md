# SolarDeye Admin Control Plane — Executive Blueprint v1

## Goal
Transform the admin area from basic management pages into a true SaaS control plane.

## Included foundation in this build
- Executive admin dashboard
- Subscribers management
- Plans and offers foundation
- Subscription lifecycle page
- Devices control center
- Device types / API integrations hub
- Services health center
- Internal mail foundation
- Finance ledger foundation
- Activity log foundation
- Roles and permissions matrix page

## Core design rules
- Admin panel never shows personal solar runtime data
- Tenant portal only shows tenant-owned data
- Device integrations are adapter-ready
- Future providers should plug into a standard internal schema
- Arabic/English must both be considered in layout and copy

## Heavy version roadmap after this build
1. Subscriber deep profile tabs
2. Ticketing with SLA and assignments
3. Quotas & credits engine
4. Offer engine and promo rules
5. Feature flags / package entitlements
6. Full i18n dictionaries
7. Email verification / password reset / invitations
8. White-label and reseller support
