# Heavy v6.2 — Support Mailbox UX

## الهدف
تحويل العمل اليومي للدعم من بطاقات كبيرة إلى تجربة Inbox / CRM أسرع، مع إبقاء مركز قيادة الدعم كلوحة مراقبة وتشغيل.

## التغييرات الرئيسية

### 1. دعم المشترك داخل الإدارة أصبح Mailbox
المسار:
`/admin/users/<id>?tab=support`

- قائمة وارد مضغوطة للرسائل والتذاكر المفتوحة.
- معاينة محادثة واحدة في الوسط بدل عرض كل البطاقات دفعة واحدة.
- صندوق رد ثابت تحت المحادثة.
- لوحة جانبية لبيانات المشترك، سير العمل، وآخر حركات التدقيق.
- الأرشيف المغلق أصبح قائمة مضغوطة بدل بطاقات كبيرة.

### 2. الردود الجاهزة داخل المحادثة
داخل كل composer يوجد زر "الردود الجاهزة" مع ثلاثة إجراءات:

- إدراج في الرد.
- إدراج + تغيير الحالة.
- إرسال مباشر بعد تأكيد المستخدم.

الحالات المقترحة تلقائياً:

- نحتاج معلومات / بانتظار ردك → waiting_user.
- تم حل المشكلة / تم إغلاق التذكرة → closed.
- تم استلام / تم تحويله للمدير المسؤول → assigned.
- تحديث متابعة → in_progress.

### 3. مركز قيادة الدعم أصبح Compact افتراضياً
المسار:
`/admin/support-command-center`

- العرض الافتراضي: قائمة مضغوطة مناسبة لعشرات العناصر.
- إضافة Toggle بين: قائمة / بطاقات.
- إبقاء الإحصائيات والفلاتر و SLA و Audit Log.
- زر "فتح في البريد" يفتح تبويب دعم المشترك على المحادثة المطلوبة.

### 4. ترجمة الحالات والأولويات
بدل عرض القيم التقنية مثل:
`open`, `waiting_user`, `high`, `normal`

تظهر بالعربية:

- open → مفتوح
- assigned → مخصص
- in_progress / pending → قيد المتابعة
- waiting_user → بانتظار المستخدم
- closed → مغلق
- normal → عادي
- high → مهم
- urgent → عاجل

### 5. إصلاح تكدس رسائل النجاح
- دمج الرسائل المتكررة في الواجهة.
- عدم عرض أكثر من رسالتين flash في نفس اللحظة.
- اختفاء تلقائي بعد وقت قصير.
- تعطيل أزرار الإرسال بعد أول إرسال لمنع الضغط المكرر.
- الحفاظ على قيمة زر الإجراء عند التعطيل، مثل assignment_only أو close.

### 6. تحسين منطق الإغلاق
عند إرسال رد جاهز للإغلاق، لا يضيف النظام رسالة إغلاق ثانية مكررة إذا كان الرد العام موجوداً بالفعل.

## فحص مقترح بعد النشر

1. افتح `/admin/support-command-center` وتأكد أن العرض الافتراضي قائمة مضغوطة.
2. جرّب تبديل العرض إلى بطاقات ثم العودة إلى قائمة.
3. افتح محادثة من زر "فتح في البريد".
4. داخل تبويب الدعم جرّب:
   - إدراج رد جاهز.
   - إدراج + حالة.
   - إرسال مباشر.
   - إغلاق مع رد جاهز.
   - اعتماد المدير.
5. تأكد أن الرسائل لا تتكرر في أعلى الصفحة.
6. تأكد أن الحالات والأولويات تظهر بالعربية.
